Vectors

Geometry of Vectors

Visually, we draw an arrow from the origin to the specified point. While this works well in two dimenstions, and somewhat in 3 dimensions, this is rather impossible to imagine in higher dimensions.

Addition

A vector with n real components is in ℝn (read: R n, not R to the n-th power). Similarly, a vector with n complex components is in ℂn.

The usual notation for vectors in printed books is a lowercase letter. In handwriting, a lowercase letter with an arrow above is commonly used.

u=u→=[ u1 u2 ] =(u1,u2)

On this site we will use the arrow-above notation, because it better distinguishes a vector from a non-bold variable name. Sometimes in floating text, the list notation in round parentheses is used.

Vector Operations

Vector addition and scaling (multiplying a scalar and a vector) are done component-wise.

Addition

[ u1 u2 ] + [ v1 v2 ] = [ u1 + v1 u2 + v2 ]

Scaling

a [ u1 u2 ] = [ au1 au2 ]
