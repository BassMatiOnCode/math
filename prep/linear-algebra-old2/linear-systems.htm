Linear Systems of Equations

Subtitle

We have a number of unknowns which are related to each other in a given way. Solving the system means finding values for the unknowns that fulfill these relations.

Examples

Example 1.1

Let's assume the following equations:

x+2y=5 4x+y=6

Solution:

This is a system of two equations with two unknowns. We already know that equations of the form

ax+by+c=0

describe a straight line in a two-dimensional coordinate system. So our sysem describes two lines. The solution is the point (x, y) that lies on both lines. In other words, we are searching for the intersection point of these two graphs:

We measure the coordinates of the intersection point and find the solution (1.5, 2).

First, we set choose x = 0 and solve equation 2 for y:

0+2y =5 y =5/2

This gives us the first point (0, 2.5). Then we choose y = 0 and slove for x:

x+2⋅0 =5 x =5

This gives us the second point (5, 0), and we can draw the blue line.

The same procedure applied to the second equation gives us the points (0, 6) and (1.5, 0) for the red line. Then we can measure out the intersection point (1, 2), wich is the solution for the equation system.

Of course, manual measurings in a diagram seldomly produce an exact solution. An exact algebraic solution uses the substitution method to solve the system:

x+2y=5 4x+y=6

There are always multiple possible routes we can follow, but here the second equation offers itself for easily solving for y:

x+2y=5 y=6−4x

Then we substitute y in the first equation with the right–hand side expression:

x+2(6−4x)=5 y=6−4x

Then we simplify and order:

x+2⋅6−2⋅4x=5 y=6−4x
x+12−8x=5 y=6−4x
x−8x=5−12 y=6−4x
−7x=−7 y=6−4x
x=−7 −7=1 y=6−4x

There we found our first solution, x = 1. We can now substitute x in the second equation with that result:

x==1 y=6−4⋅1=2

and thereby obtain our second result, y = 2. So the exact result set of the equation system is (1, 2), which is in accordance to the estimated graphical solution we obtained above.

Solutions Sets

If we analyze the graphical representation of the equation system with lines, be can imagine three different solution sets to the system:

  1. The lines intersect in a single point: The system has one unique solution, which are the coordinates of that point. The solution set contains one unique solution.
  2. Both lines are parallel to each other with some distance between them: The system has no solutions, because parallel lines never intersect anywhere. The solution set is empty.
  3. Both lines are parallel with no distance between them: There are infinitely many solutions, because every point on the first line is also a point on the second line. The solution set contains infinitely many solutions..

Example 1.2

Question: For what values of the unknowns x, y and z are the following equations satisfied?

2x+2y+5z =11 4x+6y+8z =24 x+y+z =4

Solution:

This is an equation system with three variables (unknowns). A geometrical solution is rather impractial, because we would have to draw three dimensions on a two-dimensional sheet of paper. So we will apply the substitution method as before.

We can solve the third equation for z and substitute z in the second equation with the result:

4x+6y+8(4−x−y) =24 z =4−x−y

The second equation simplified, ordered and solved for y:

4x+6y+8⋅4−8⋅x−8⋅y =24 4x−8x+6y−8y =24−32 −4x−2y =−8 −2y =+4x−8 y =−2x+4

Then we can substitute y and z in the first equation:

2x+2(−2x+4)+5(4−x−y) =11

Simplified, ordered and solved for x:

2x+2(−2x+4)+5(4−x−y) =11 2x+2⋅−2x+2⋅4+5⋅4−5x−5y =11 2x−4x+8+20−5x−5y =11 2x−4x−5x−5y =11−8−20 =−17 −7x−5y =−17

We must substitute the y that came in with the substitution of z:

−7x−5(−2x+4) =−17 −7x−5⋅−2x−5⋅+4 =−17 −7x+5⋅2x−5⋅4 =−17 +3x−20 =−17 +3x =−17+20 =3 x =1

This is our first result, which we can substitute into the solution for y above:

y =−2x+4 y =−2⋅1+4 y =2

Our second result. Now we can sove the equation for z by substituting x and y with their solutions:

z =4−x−y z =4−1−2 z =1

This is the solution for the third variable.

So the solution set for the equation system has a single unique solution (1, 2, 1).

Verification:

We subsitute the variables for their results in the original equation system:

2x+2y+5z =11 4x+6y+8z =24 x+y+z =4

Let's start with the first equation:

2x+2y+5z =11 2⋅1+2⋅2+5⋅1 =11 2+4+5 =11 11 =11

That's correct. Equation 2:

4x+6y+8z =24 4⋅1+6⋅2+8⋅1 =24 4+12+8 =24 24 =24

Also correct. Equation 3:

x+y+z =4 1+2+1 =4 4 =4

Everything is correct. The solution (1, 2, 1) has been verified to satisfy the equation system.

Example 1.3

The substitution method incorporates a fair amount of copying and transformation steps, which easily can introduce errors. In this example we will apply a different method called "elimination of variables", which involves a different kind of transformation: Adding multiples of one row to another.

Here again the original equation system:

2x+2y+5z =11 4x+6y+8z =24 x+y+z =4

The first step is to eliminate the 2x in row 1. For this purpose, we replace row 1 with the sum of row 1 minus 2 times row 3. In short:
R1 − 2×R3 → R1:

2x+2y+5z =11 x+y+z =4 (2−2⋅1)x +(2−2⋅1)y +(5−2⋅1)z =11−2⋅4 (2−2)x +(2−2)y +(5−2)z =11−8 0x +0y +3z =3 z =3÷3 =1

Fortunately, this also eliminates the 2y and leaves only the variable z. So we have our first result: z=1.

The next goal is to eliminate the 4x in row 2. So we replace row 2 with row 2 minus 4 times row 3:
R2 − 4 × R3 → R2:

4x+6y+8z =24 x+y+z =4 (4−4⋅1)x+(6−4⋅1)y+(8−4⋅1)z =24−4⋅4 (4−4)x+(6−4)y+(8−4)z =24−16 2y+4z =8

The current state of the system:

z =1 2y+4z =8 x+y+z =4

Now we can subtract 4 times row 1 from row 2 to eliminate the 4z in row 2. At this point, the procedure should be clear, so we left out the intermediary steps. We still recommend to use a scratchpad and write down these steps until your brain is used to juggle around all these numbers and signs and additions and subtractions.

z =1 2y =4 x+y+z =4

Then divide row 2 by 2, and subtract row 1 from row 3, which should eliminate the z in row 3:

z =1 y =2 x+y =3

Finally subtract row 2 from row 1 to eliminate the y:

z =1 y =2 x =1

So we have it. The solution of the equation system is (1, 2, 1), the same result as above.

It appears that we have much less copying and computation effort with this method, especially if you can do the calculations in your head.

Geometrical Interpretation

An equation of the form ax+by+cz=d describes a plane in a three-dimensional space. Three planes may intersect in a point, the system of equations has one unique solution. Two planes may intersect in a line. If the plancs are parallel with some distance between them, they will not intersect. If the planes are identical, they intersect everywhere.

Before carrying on, we need to set up some terms.

The definition specifies coefficients and variables in ℂ, the set of complex numbers. And because the set of real numbers ℝ is a subset of the complex numbers (ℝ ⊂ ℂ), a linear equation works perfectly with real numbers.

We can derive a number of facts from the definition 1.1:

  1. Variables only appear to the power of one: xi1=xi

There are no squares, cubes or higher exponents or roots.

  1. There cannot be fractions of in the form of xixj

The variable in the denominator would have a negative exponent: xj−1, which we already have ruled out above.

  1. There cannot be products of variables: xi⋅xj

The operator between two variables must be additive, either + or −. Multiplicative operators ( ⋅ or / ) are not allowed.

  1. There cannot be nonlinear functions of variables, such as sinx,cosx,ax, et cetera.
  2. If a coefficient is missing, its value is 1.
  3. If a variable is missing, its coefficient has a value of 0.
