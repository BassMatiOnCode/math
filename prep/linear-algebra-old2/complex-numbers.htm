If we define a complex number as an ordered pair of numbers, we can identify each complex number as a point in a plane.

Suppose a number line in a two-dimensional plane. Multiplying a real number a with −1 on the number line can be interpreted as a rotation of a vector a→ with the magnitue ||a→||=a by 180 degree around the origin (0, 0):

Now let's apply this concept of rotation to the multiplication of a real number a and the imaginary unit i. We know that a⋅i⋅i=−a, because i⋅i=−1. So we can safely conclude that if a multiplication by minus one corresponds to a rotation of a vector by 180 degree, and two times a multiplication with i also corresponds to a rotation by 180 degree, then one times a multiplication with i must be a rotation by 180/2 = 90 degree:

Now that we have a complete number system at hand, we can turn our focus to the

Fundamental theorem of algebra

Without complex numbers, we were not able to solve polynomial equations if the form of a2=&minus1. This is now history.

The proof of the theorem is rather difficult and does not provide any hint to how these equations can be solved. We can use numerical approximation techniques to get sufficiently precise solutiones. However, for polynomials of low orders (n < 5) there are formulas we can use.

"Rationalizing" a Complex Denominator

Consider the following fraction:

a+bi c+di

To eliminate the complex denominator, we expand with the complex conjugate of the denominator:

a+bi c+di c−di c−di = (a+bi)(c−di) (c+di)(c−di)

Application of the distributive law obtains
