Set Theory

Notations

SET: A collection of objects. Sets are designated with uppercase letters (A,B,C,…). A set may contain elements, or not, in which case it is said to be empty. An empty set is also called the null-set.

ELEMENT of a set: An element is an object. It may be a member of a specific set. Elements are designated with lowercase letters (a,b,c,…). If an element a is an element of a set A, we write a∈|.
