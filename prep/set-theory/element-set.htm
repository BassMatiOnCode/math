Element, Set and Relations

Set theory is about working with elements. So we first need to define what an element is:

Elements can be "grouped" into sets, usually for the purpose of addressing elements with common properties.

Sets can be defined by enumerating elements in curly braces:

A={a,b,c}
A={1,2,3,…}
A={1,2,4,8,…}

An ellipsis indicates that there are infinitely many elements following, according to the implicit rule inherent in the sequence of precding elements.

A set can also be defined by defining conditions that must be fulfilled by all elements of the set:

A={a∈A|a∈ℕ,3≤a≤9}

Read: The set A contains all numbers from the natural numbers between 3 and 9, inclusive.

Then we need a way to say that an elment is a member of a set:

Set Membership

This is also read as "element a is contained in A" or "A contains element a".

Sets can be contained in other sets:

Definition of subset

A is completely contained in B. No element of A is outside of B. A may contain all elements of B, i. e. A = B. This can also be read as "B is a superset of A".

Definition of proper subset

An intuitive image should make that clear:

A is completely contained in B, no element of A is outside of B, but there must be some members of B that are not member of A.

Intersection

The intersection of A and B contains all the elements that are members of both, A and B.

Union

The union of A and B contains all elements that are members of A or members of B or members of both, A and B. The operator "or" here is the logical inclusive OR operator, not the exclusive OR.

The difference of A minus B contains all elements of A that are not member of B.

Notation

Some notation conventions:

{ } or ∅
designate the empty set
a ∈ A
a is a member of the set A
A = B
The set A is equal to the set B. All members of A are also members of B, and vice versa.
A ⊂ B
A is a proper subset of B
A ⊆ B
A is a subset of or identical to B

Some ways to define a set:

List of elements:
A={1,2,3}

A contains the elements 1, 2 and 3.

By rule:

A={a|a∈ℕand1≤a≤3}

A contains the natural numbers 1, 2 and 3.

Examples

Let A = { 0, 1, 3 } and B = { 0, 1, 2, 4 }. Then
