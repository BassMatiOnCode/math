Euclician Geometry

Euclid defined 5 postulates as the foundations of geometry. None of these postulates can be proven from each other, they are independent from each other. With these postulates, all other theorems in geometry can be proven. They are:

  1. Any two points can be connected by a straight line.
  2. Any line segement can be stretched out forever in either direction.
  3. We can always define a circle given a center and a radius.
  4. All right angles are congruent.
  5. For any line and any point not on that line there is exactly one line parallel to the given line that passes through that given point.

Euclid introduced the concept of proof.
