Matrix

Introduction

An m × n matrix Mat( m, n ) is a rectangular array of entries, m rows high and n columns wide.

A = [ a1,1 a1,2 a1,3 a2,1 a2,2 a2,3 ] A matrix is denoted with an uppercase letter.

Addition

Matrices are added component-wise:

A+B =[ a1,1 a1,2 a1,3 a2,1 a2,2 a2,3 ] + [ b1,1 b1,2 b1,3 b2,1 b2,2 b2,3 ] = [ a1,1+b1,1 a1,2+b1,2 a1,3+b1,3 a2,1+b2,1 a2,2+b2,2 a2,3+b2,3 ]

Scaling

Similar to the addition, a matrix is multiplied (scaled) component-wise with a value:

n⋅A =n⋅ [ a1,1 a1,2 a1,3 a2,1 a2,2 a2,3 ] =[ na1,1 na1,2 na1,2 na2,1 na2,2 na2,2 ]

Matrix Multiplication

The multiplication of two matrices A × B = C is more complicated. Row elements of matrix A are multiplied with column elements of matrix B and then summed up to form the result element in matrix C.

Not all matrices can be multiplied with each other; the width of A must be equal to the height of B. The dimensions of the result C is defined by the height of A and the width of B.

For manual calculations, it helps to arrange the matrices in the following way:

[B] [A] [C]

An example. The result matrix C is at the bottom right. To the left is matrix A, above is matrix B:

Click on a matrix C cell. The result cell determines which row of A is multiplied with which column of B, the cell sits at the intersection of the row and colulum If A is a matrix of size m×n and B of n×p, a result cell value can be calculated after the following formula:

ci,j= ∑k=1nai,k⋅bk,j

with 1 ≥ i ≥ m and 1 ≥ j ≥ p

For example, c1,1 is   2 ⋅ 1 − 1 ⋅ 3 = −1.

Using the suggested notation layout has another advantage when you have to calculate a chained matrix multiplication expression, such as A ⋅ B ⋅ C. The result of becomes the left operand of (A ⋅ B) ⋅ C, so you can write C to right side of B:

  B C
A A⋅B (A⋅B)⋅C

If the product becomes the left operand of the next chained multiplication, you can apply the scheme downwards. For example, A⋅(B⋅C) can be arranged as

  C
B B⋅C
A A⋅(B⋅C)

Non-Commutativiy

Matrix multiplication is not commutative:

A⋅B≠B⋅A

Associativity

Matrix multiplication is associative:

(A⋅B) ⋅C=A⋅(B⋅C)

Neutral Matrix of Multiplication

A conclusion of the two sections above: The neutral element of matrix multiplication has ones at the principal diagonal, and zeroes at all other positions. Such a matrix is called identity matrix:

[ a1,1 a1,2 a1,3 a2,1 a2,2 a2,3 a3,1 a3,2 a3,3 ] ⋅ [ 1 0 0 0 1 0 0 0 1 ] = [ a1,1 a1,2 a1,3 a2,1 a2,2 a2,3 a3,1 a3,2 a3,3 ]

Note that the size of the identity matrix must match the other matrix operand.

Inverse Matrix

A matrix A multiplied with its inverse matrix evaluates to the identity matrix:

A⋅A-1=I A-1⋅A=I

An invertible matrix has both a left and a right inverse, and they are identical.

Note that not all matrices have an inverse!

Inverse of Product of Matrices

(A⋅B)-1 = B-1⋅A-1

Transpose

The transpose AT of the matrix A is obtained by interchanging rows and columns:

[ a1,1 a1,2 a1,3 a2,1 a2,2 a2,3 a2,1 a2,2 a2,3 ]T = [ a1,1 a2,1 a3,1 a1,2 a2,2 a3,2 a1,3 a2,3 a2,3 ]

The transpose of a single-row matrix is a vector.

Transpose of Product

(A⋅B)T = BT⋅AT

Diagonal

The main diagonal of a square matrix is the line of cells from the top left to the bottom right corner.

A diagonal matrix has non-zero values on the main diagonal, all other entries above and below are zero.

Symmetric

A symmetric matrix is equal to its transpose:

A=AT

Antisymmetric

An antisymmetric matrix is equal to the negative transpose:

A=−AT

Both symmetric and anti-symmetric matrices are square.

Triangular Matrix

An upper triangular matrix has zeroes below the main diagonal. Similarly, a lower triangular matrix has zeroes above the main diagonal.

If A and B are upper triangular matrices, then the product of A and B is also an upper triangular matrix.
