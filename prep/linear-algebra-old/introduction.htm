Linear Equations

Consider the following equation:

a1x1+ a2x2+ ⋯+ anxn= b

This is a linear equation with n∈ℕ terms. Each term consists of a coefficient aj and a variable or indeterminant or unkown xj with j∈{j∈ℕ|1≤j≤n}. On the right side we find a number, the solution b. All these elements are numbers ∈ℂ, and since ℝ⊂ℂ, linear equations should work with both real or complex numbers.

If a variable xj doesn't appear in the equation, the associated coefficient aj is zero. If a cofficient bj doesn't appear, its value is one. Coefficients may take negative values, then the operator on the left side is a minus instead of a plus.

The left side of the equation is not a polynomial. Variables in linear equations are always to the power of one (xj1). There are no other exponents or radicals. Additionally, there are no products or quotients of variables, no functions like sine or cosine or exponential functions or whatever.

These are examples that are not linear equations:

3x+y=0 (1) 3x+1y=0 (2) 3ex−πy=3 (3) 3x+yz=4 (4) 3x+yz=4 (5)

Standard Form

The standad form of an equation has the unknowns on the left side of the equal sign, and a constant value on the right. Sometimes an equation is not in the standard from, but it can be transformed into the standard form. An example:

2x1 − x2 + 6 = 13 − 4x1

We subtract 6 from both sides of the equation:

2x1 − x2 = 7 − 4x1

Then we add 4x1 to both sides:

6x1 − x2 = 7

Which gives us the standard form.

Solutions of Linear Equation Systems

Usally we want to solve such as system for the unknowns, that is determine a solution, a set of values for the variables xj that satisfies the entire system of equations.

Solutions are often written as ordered tuples (s1,s2, ⋯ ,sn), where sj is the solution value for the corresponding unknown xj.

For example, (3,4,−5) is a solution for the equation

x1−4x2−6x3=17

Proof: We plug in the values of the solution into the corresponding variables:

(3)−4(4)−6(−5) =? 17 3−16+30 =? 17 17 =? 17 true

It turns out that the solution is correct, otherwise we would have a contradiction in the last row.

Example for a tuple that is not a solution to equation system: (3,4,5)

(3)−4(4)−6(5) =? 17 3−16−30 =? 17 −43 =? 17 false

The set of all solutions of a linear equation system is called the solution set. A solution set can be empty, or have one or infinitely many solutions.

A system with no solutions is called inconsistent. A consistent system can have one or more solutions.

Systems of linear equations are equivalent if the have the same solution set.

Example: (−3,−8,4) is not a solution of the equation system

2x1 − x2 + 1.5x3 = 8 x1 − 4x3 = −7

Proof: We substitute the solution values into the first equation:

2(−3) − (−8) + 1.5(4) = 8 −6 + 8 + 6 = 8 8 = 8 true

As it turns out, the tupel is actually a solution for the first equation. Then we subsitute the solution into the second eqution:

(−3) − 4(4) = −7 −3 − 16 = −7 − 19 = −7 false

The last equation contains a contradiction, so the suggested solution is not a solution of the entire equation system.
