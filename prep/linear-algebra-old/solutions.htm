Solutions

The reason why we transform a system of linear equation or its matrix into echelon or reduced echelon form is that we can easier analyze its solutions.

If a linear system has free variables, basic variables are expressed in terms of free variables. You can choose arbitrary values for the free variables. Your choice then determines the basic variables.

A consistent system has one unique solution if there are no free variables. A consistent system has infinitely many solutions if there is at least one free variable.

Solving a System of Linear Equations

To solve a system of linear equations, follow these steps:

  1. Create the augmented matrix from the equations
  2. Reduce the matrix to the echelon form
  3. Rewrite the matrix in equation form
  4. Solve each equation for the basic variable

Describing a Solution

Sometimes not a complete solution is required, and is sufficient to describe the solution or point out certain properties of the solution:

  • Does the system have at least one solution?
  • Does the system have a single unique solution?

These questions can be deduced from the echelon form, the reduced echelon form is not necessary.

If the echelon form has a row with a contradiction (e. g. 0 = 5), the system has no solution and is inconsistent.
