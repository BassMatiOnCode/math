Echelon Form of a Matrix

Definition

The term echelon form defines precisely what we previously have called the "triangular form" of a matrix. But firstly, we will need a definition:

The leading entry is the first non-zero element in a row of a matrix .

A matrix is in echelon form if these two conditions are fulfilled:

  1. Rows of all zeroes are below all non-zero rows.
  2. Each leading entry has only zeroes in the column below.

Because a leading entry has per definition 1 all zeros in front of it, any leading entry must be in a column further right to the leading entry of the row above. If a leading element has non-zero elements below, the matrix is not in echelon from. If a leading element has a non-zero element in front, the matrix is not in echelon form. Some columsn may not have a leading entry, but the matrix can still be in echelon form.

When a matrix is in echelon form, the leading element in a row is called pivot.

A column that contains a pivot element is called a pivot column.

Reduced Echelon Form

The reduced echelon form of a matrix is very useful because its even simpler to obtain the solution values for the unknowns. In fact, we already have used that form in the previous example.

So, here is the definition:

A matrix is in reduced echelon from if the following conditions are satisfied:

  1. The matrix is in echelon form
  2. The leading entry of each row is 1
  3. The entries above each leading entry are all zero

Effectively, we have zero entries in front, below, and above the leading entries, and each leading entry is in a further right column than the leading entry in the column above.

Uniqueness of the Reduced Echelon Form

A system of equations or its autmented matrix may have multiple different but equivalent echelon forms. But a system can have only one unique reduced echelon form. Equation systems that can be reduced to the same reduced echelon form are equivalent.

Reduction Algorithm

To reduce a matrix to echelon form, follow these steps:

  1. Bring a row with a pivot in the first column into the first row.
  2. Replace rows below to eliminate entries in the pivot column.
  3. Move to the next pivot column and repeat from step 2.

Position of Pivots

Even though matrices can have different echelon forms, the pivots will always be in the same locations.

Examples

Example 1

Bring the following matrix into echelon form:

[ 0 1 2 0 -1 -2 2 0 3 2 2 8 20 0 6 ] ~ [ -2 2 0 3 2 0 1 2 0 -1 2 8 20 0 6 ] ~ [ -2 2 0 3 2 0 1 2 0 -1 0 10 20 3 8 ] ~ [ -2 2 0 3 2 0 1 2 0 -1 0 0 0 3 18 ]

Example 2

Reduce the following matrix:

[ 1 4 8 2 8 -1 -3 0 0 1 -4 16 ] ~ [ 1 4 8 0 0 -17 0 12 24 0 -8 8 ] ~ [ 1 4 8 0 12 24 0 -8 8 0 0 -17 ] ~ [ 1 4 8 0 12 24 0 0 24 0 0 -17 ] ~ [ 1 4 8 0 12 24 0 0 24 0 0 0 ]

Example 3

Bring the result matrix of the previous example into reduced echelon form.

[ 1 4 8 0 12 24 0 0 24 0 0 0 ] ~ [ 1 4 8 0 1 2 0 0 1 0 0 0 ]

This is rather simple, because we can scale the rows down to make the pivot cells zero.

Example 4

Transform the following matrix to reduced echelon form:

[ 1 2 3 4 4 5 6 7 6 7 8 9 ] ~ [ 1 2 3 4 0 -3 -6 -9 0 -5 -10 -15 ] ~ [ 1 2 3 4 0 1 2 3 0 1 2 3 ] ~ [ 1 0 -1 -2 0 1 2 3 0 0 0 0 ]

The matrix has only two pivot columns and cannot be reduced further.
