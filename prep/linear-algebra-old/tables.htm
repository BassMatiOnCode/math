Definition 1.7.1: Linear Independence

An indexed set of vectors {v→1, ⋯ ,v→m} in ℝn is linearly independent if the vector equation

x1v→1+⋯+xmv→m=0→

has only one trivial solution. If there are one or more non-trivial solutions, the set of vectors is linearly dependent.

Theorem 1.7.1   Linearly Dependent Set

An indexed set {v→1, ⋯ ,v→n} of non-zero vectors with n≥2 is linearly dependend if and only if at least one of the vectors is a linear combination of the others.

In particular, if v→1≠0→ and some v→i with i > 1 is a linear combination of the preceding vectors v→1, ⋯ v→i−1.

Theorem 1.7.2 More vectors than dimensions

Any set {v→1, ⋯ ,v→n} in ℝm is linearly dependent if n>m.

Theorem 1.7.3   Set with 0→ is Linearly Dependent

If a set S={v→1, ⋯ ,v→n} in ℝm contains the zero vector 0→, then the set is linearly dependent.
