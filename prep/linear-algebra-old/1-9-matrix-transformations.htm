1.9 Matrix of a Linear Transformation

Example 1

Standard Transformations

[ 1 0 0 1 ] Identity matrix, does not change original.
[ 1 0 0 −1 ] Reflection through x1 axis.
[ −1 0 0 1 ] Reflection through x2 axis.
[ 0 1 1 0 ] Reflection through x2=x1.
[ 0 −1 −1 0 ] Reflection through x2=−x1.
[ −1 0 0 −1 ] Reflection through the origin (0,0).
[ k 0 0 1 ] Horizontal scaling.
[ 1 0 0 k ] Vertictal scaling.
[ 1 k 0 1 ] Horizontal shear.
[ 1 0 k 1 ] Vertical shear.
[ 1 0 0 0 ] Projection onto the x1 axis.
[ 0 0 0 1 ] Projection onto the x2 axis.
