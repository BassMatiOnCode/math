1.8 Transformations

We know now that a matrix equation Ax→=b→ is a different notation for a vector equation x1x→1+ ⋯ +xnx→n=b→. However, a matrix A can be seen as transformation that transforms a vector x→ into a new vector b→. The matrix can also be seen as a function that takes x→ as input and produces the vector b→ as output.

A system of linear equations:

a1,1x1,1 + ⋯ + a1,nx1,n = b1 a2,1x2,2 + ⋯ + a2,nx2,n = b2 ⋮ ⋮ ⋮ ⋮ am,1xm,1 + ⋯ + am,nxm,n = bm

A matrix equation:

Ax→=b→

A vector equation:

x1a→1 + ⋯ + xna→n = b→

A transformation or function or mapping from a vector x→ in ℝm to a different vector b→ in ℝp

A:ℝn→ℝm
A[ x1 x2 x3 ] = [ b1 b2 ]

The correspondence from x→ to Ax→ is a function from one set of vectors to another set of vectors. This is a generalization of the function concept from numbers arguments to vector arguments.

The set ℝm is called the domain of the transformation, the set ℝn is called the codomain.

The transformation assigns an image in the codomain to every vector in the domain of the tranformation. The set of all images in the codomain is called the range of the transformation.

Or a projection from one space to another:

A:ℝn↦ℝm

Matrix Transformations

In this chapter we will focus on mappings associated with a matrix multiplication x→↦Ax→. For each x→ in ℝn, T(x→) is computed as Ax→, where A is a matrix of size m × n, the vector x→ is a matrix of size n × 1, and the result vector b→ is a matrix of size m × 1.

Thus the domain of T is ℝn as the number of columns in the matrix A, and the codomain of T is ℝm as the number of rows in the matrix A.

The range of T is the set of all linear combinations of the columns of A, because each image T(x→) is of the form Ax→=x1a→1+ ⋯ +xna→n.

Example 1

Let A= [ 1 −3 3 5 −1 7 ] , u→= [ 2 −1 ] , b→= [ 3 2 −5 ] , c→= [ 3 2 5 ]

Define a transformation T : ℝ2↦ℝ3 so that

T(x→) = Ax→ = [ 1 −3 3 5 −1 7 ] [ x1 x2 ] = [ x1 − 3x2 3x1 + 5x2 −x1 + 7x2 ]

Find T(u→), the image of u→ under the transformation T.

Solution:

T(x→) = Ax→ = [ 1 −3 3 5 −1 7 ] [ 2 −1 ] = [ (1×2) + (−3×−1) (3×2) + (5×−1) (−1×2) + (7×−1) ] = [ 2+3 6−5 −2−7 ] = [ 5 1 −9 ]

Exercise 1.2: Find an x→ in ℝ2 whose image under T is b→.

Solution: Sove T(x→) for x→:

T(x→) = Ax→ = [ 1 −3 3 5 −1 7 ] [ x1 x2 ] = [ 3 2 −5 ]

We create an augmented matrix and reduce:

[ 1 −3 3 3 5 2 −1 7 −5 ] ~ [ 1 −3 3 0 14 −7 0 4 −2 ] ~ [ 1 −3 3 0 1 −1/2 0 1 −1/2 ] ~ [ 1 0 3/2 0 1 −1/2 0 0 0 ]

The solution is x→= [ x1 x2 ] = [ 32 −12 ]

The image of this vector x→ under T is the given vector b→.

Exercise 1.3: Is there more than one x→ whose image under T is b→?

Solution: The previous answer showed that Ax→ has a unique solution. So there cannot be another vector x that also solves the equation.

Question 4: Determine if c→ is in the range of the transformation T.

Answer: The vector c→ is in the range of T if c→ is the image of some x→ in ℝ2, or in other words, if c→=T(x→) for some x→. This is just another way of asking if the system Ax→=c→.

Answer: We create a matrix and reduce:

[ 1 -3 3 3 5 2 -1 7 5 ] ~ [ 1 -3 3 0 14 -7 0 4 8 ] ~ [ 1 -3 3 0 1 -1/2 0 1 1/2 ] ~ [ 1 -3 3 0 1 -1/2 0 0 1 ]

The last row in the last matrix has a contradiction (0=1), which shows that the equation system is inconsistent and therefore has no solution. So c→ is not in the range of T.

The question 1.3 is a uniqueness problem for a system of linear equations, translated into the language of matrix transformations. Similarly, question 1.4 is an existence problem, does there exist an x→ whose image is c→?

The following transformation matrix has an interesting geometrical explanation:

If A=[ 1 0 0 0 1 0 0 0 0 ] ,

then the transformation x→↦Ax→ projects points in ℝ3 onto the x1x2 plane, because

[ x1 x2 x3 ] ↦ [ 1 0 0 0 1 0 0 0 0 ] [ x1 x2 x3 ] = [ x1 x2 0 ]

Whatever you will put in for x→, its x3 component will be diminished to zero.

Example 3: Shear Transformation

Let A=[ 1 2 0 1 ] .

The transformation T:ℝ2↦ℝ2 defined by T(x→)=Ax→ is called a shear transformation. Let this transformation act on the points (0,0), (0,2), (2,2) and (2,0), the corners of a square.

So we perform the transformations:

T(u→1) = [ 1 2 0 1 ] [ 0 0 ] = [ 0 0 ]
T(u→2) = [ 1 2 0 1 ] [ 0 2 ] = [ 4 2 ]
T(u→3) = [ 1 2 0 1 ] [ 2 2 ] = [ 6 2 ]
T(u→4) = [ 1 2 0 1 ] [ 2 0 ] = [ 2 0 ]

The result is a parallelogram with the points (0,0), (4,2), (6,2) and (2,0);

Linear Transformations

Theorem 5 in section 1.4 showed that if A is a matrix of size m × n, the transformation x→↦Ax→ has the properties

A(u→+v→)=Au→+Av→ and A(cu→)=cAu→

for all u→, v→ in ℝn and all scalars c. Written if function notation:

Every matrix transformation is a linear transformation, but not all linear transformations are matrix transformations.
