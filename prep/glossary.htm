Glossary of Terms

Abelian Group

An abelian group is a commutative group.

Abelsche Gruppe
Abelian Group
Field

A field is a set with the two operations addition and multiplication, where (K, +) is an aebelian group with 0 as the neutral element, and (K, &mdot;) is an abelian group with 1 as the neutral element, and they are distributive.

Group

A combination of a set and an operation. The operation is associative and has an identity element, and every element of the set has an inverse element.
