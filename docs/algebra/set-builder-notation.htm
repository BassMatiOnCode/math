Set-Builder Notation

Introduction

We could easily define the set of natural numbers by enumeration:

S={1,2,3,…}

But this set is not very well defined. Consider this one:

S={3,5,7,11,13,…}

One could guess that it is the set of primes. But that is just a guess.

We need predicates.

If P(x) is a predicate, then we can form the set that contains all entities of a such that a is in the domain of discourse for P and P(a) is true.

The notation {x | P(x)} is used to denote this set, this is the intensional definition of the set. It is read "the set of x such that P(x)". The name of the variable (x) is arbitrary and can be chosen freely.

NOTE that the domain of discourse of P must be a set!

Example: If the predicate P(x) is "x is an even number", and if the domain of discourse of E is the set ℕ+0, then this is a definition for the set of even numbers:

{x|E(x)}={0,2,4,6,…}

Sometimes it is useful to declare the domain of discourse explicitly:

{x∈ℕ+0|E(x)}={0,2,4,6,…}

Or more generally:

{x∈D|P(x)}

Here, D is a set and P is a predicate whose domain of discourse includes all elements in D. The definition can be read as "the set that contains all entities x that are members of the set D for which P(x) is true".

An equivalent notation is

{x|x∈D∧P(x)}

If the predicate P(x) is a conjuction P1(x) ∧ P2(x), the set definition is sometimes written as

{x∈D|P1(x),P2(x)}

Example: Rigorous definition of the prime numbers:

{p∈ℕ|(p>1)∧∀a∀b((a∈ℕ∧b∈ℕ∧p=ab)⇒(a=p∨b=p))}

This is read as "a set of all elements p in ℕ such that p > 1, and for all elements a and b in ℕ and all products of a times b that are equal to p implies either a is p or b is p". So if we can find a product ab where neither a nor b is equal to p, then p is not a prime number.
