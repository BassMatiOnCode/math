We can assume that b>0, because if b<0, we can rewrite the equality

a=bq+r

as

a=(−b)(−q)+r

without changing the statment. We just have to verify that −b>0.

Given b>0 and a, there are integers q1 and r1≥0 such that a=bq1+r1. For example, let's choose q1=0 and r1=a first. Then

a=bq+r a=(b)0+a a=a

Which is obviously true. If a<0, we choose q1=a and r1=a−ab:

a=bq+r a=ba+a−ab a=a

Which also is obviously true.

Now let q and r be such a pair of numbers for which r is nonnegative and minimal. If r < b, we have an Euclidean division, as per definition in the theorem. Thus, we have to prove that, if r ≥ b, then r is not minimal, we have

a=bq+r−b+b a=b+bq+r−b a=b(1+q)+(r−b)

with 0 ≤ (r − b) < r, and r is not minimal.
