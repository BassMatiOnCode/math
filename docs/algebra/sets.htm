Set

An Unordered Collection of Anything

Introduction

A set is a collection of different, unique objects:

A={a0,a1,a3,a4} B={1,2,3,4} C={□,△,▱,○} D={ {5,−2}, {−4,−3}, {5,−0} }

Set names are written in uppercase italic (oblique) letters. Set member names are written in lowercase italic letters. Literals, such as numbers or text, are written with upright characters.

A set member can be anyhing, including another set.

Set Notation

Roster Notation

In roster notation, sets are defined by the enumeration of members, enclosed in curly braces:

A={a0,a1,a3,a4}

A large number of elements can be described by an ellipsis, especially if a set describes a sequence of elements. For example, the following set contains the integer numbers from 1 to 1000:

{1,2,3,…,1000}

The size of a set is defined by the numbers of members. A set can have a limited size. An unlimited size is indicated by an ellipsis at the beginning or the end of a list. For example, the following set comprises all natural numbers:

{1,2,3, …}

This set contains all negative and positive integer numbers:

{… ,−3,−2,−1,0,1,2,3, …}

Semantic Definition

Semantic definition uses e rule to define the elements of the set:

Let A be the set whose members are the first four positive integers.

Set–Builder Notation

The set-builder notation specifies a condition which an element e must fulfill to be a member of the designated set:

A={ e|condition } A={ e:condition }

This reads as "A is the set of all elements e such that e fulfills the condition.

An example:

B={ n|n∈ℕandn<5 }

Reads: B is the set of all elements n such that n is in the set of natural numbers, and n is less than five. Which again is the set B from above.

Membership

The operator ∈ is the membership operator. It reads "is element of", or "is in" :

2∈B a3∈A

An element can be a member of multiple sets:

e∈Aande∈B

To express that an element is not in a set, we use the operator ∉, which reads "is not in" or "is not an element of" :

e∉A

Empty Set

A set with no members is the empty set:

∅={}

Uniqueness of Members

A set is defined by its unique members. So these sets are actually identical:

{1,6,3,3,1,6} = {1,6,3,1}

Definition

A set is defined by its unique members.

Equality

A set is defined by its unique members. Two sets are identical if they have the same unique members:

{1,5,27,B} = {B,27,1,5}

This also means that there is exactly one unique empty set:

{} = {}

To express that two sets are not equal, we write

{1,2,3 }≠{ 1,2,4 }

Subset

If all members of the set B are also members of the set A, then B is a subset of A:

B={ b|b∈A }

This is written as "B is a subset of A":

B⊆A

or equivalently: "A is a superset of B":

A⊇B

A set is always a subset and a superset of itself:

A⊆A A⊇A

The empty set ∅= { } is also a subset of A because there is no member of ∅ that is not a member of A.

∅⊆B

Proper Subset

If all members b of the set B are also members of the set A, but there are some members of A that are not in B, then B is a proper subset of A:

B={ b|b∈A and∃ a∈A anda∉B }

Or alternatively

B={ b|b∈A and A≠B }

This is written as "B is a proper subset of A":

B⊂A B⊊A

or equivalently: "A is a proper superset of B":

A⊃B A⊋B

Set Operations

Complement

With a given set U, which contains all elements of interest, the complement of a set A is the set of elements that do not belong to A:

A∁={e|e∈U∧e∉A}

Union

A={a|a∈B∨a∈C} A=B∪C

Intersection

A={a|a∈B∧a∈C} A=B∩C

Difference

A={a|a∈B∧a∉C} A=B−C A=B∖C

Symmetric Difference

A=(B∖C) ∪ (C∖B) A=BΔC

Cartesian Product

The cartesian product is the set of all ordered pairs (a,b) such that a∈A and b∈B:

C={(a,b)|a∈A∧b∈B} A=B⨯C

An example:

{a,b} ⨯{1,2,3}= { (a,1), (a,2), (a,3), (b,1), (b,2), (b,3) }

Well–Known Sets

The natural numbers are bacially the counting numbers:

ℕ={1,2,3,…}

Some people include the zero in ℕ, others write ℕ0 if the zero is included in the set.

Then we have the integer numbers, which extend the natural numbers to the other side of zero:

ℤ={…,−3,−2,−1,0,1,2,3,…}

The set of rational numbers or fractions of integers:

ℚ={ab|a,b∈ℤ,b≠0 }

The set of real numbers is the union of all rational numbers ℚ and irrational numbers. The correct construction of the set of real numbers is cover in analysis.

The set of complex numbers ℂ with

ℂ={a+bi|a,b∈ℝ,i=−1}

Each set above is a superset of the previous set:

ℕ⊂ℤ⊂ℚ⊂ℝ⊂ℂ

The cartesian products of these standard number sets are often used and have a shortcut notation. For example:

ℤ⨯ℤ=ℤ2 ℝ⨯ℝ⨯ℝ=ℝ3
