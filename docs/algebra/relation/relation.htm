Definition of a Relation

Suppose we have two sets, A and B. The cartesian product relates all elements ai from A to all elements bj from B. This creates a set of all possible ordered pairs (a, b):

We call it relation, because the set R relates elements from A to elements from B.

Now we can define an additional condition that the elements in R must fulfill to make it into R. So the R is a subset of the crossproduct:

R⊆A⨯B

In order to express that a pair (a, b) satisfies the relation R, or not, we can use the operator notation:

For example, if the condition were "equal", we could use the equal sign "=" to designate the relation. Then we get the well-known operator notation

a=b

Example: Equality operator.

For example, with a given set S, we could create a subset R which contains pairs of equal elements:

S={1,2,3} R={(a,b)∈S⨯S|a=b}

While S contains all possible combinations,

S⨯S={(1,1),(1,2),… ,(3,2),(3,3)}

R only contains pairs of elements that are equal:

R={(1,1),(3,2),(3,3)}

Composition

Composition of relations follows the scheme of composition of functions, where the innermost (rightmost) function is evaluated first, then followed by the evaluation of the left (outer) function:

f∘g=f(g(x))

The definition states that there must be an (intermediate) element b that connects the first relation to the second.

Example: Consider the sets A, B and C, and the relations R and S:

A={1,2,3}, B={a,b,c}, C={A,B,Cs}

… with the following relations:

 R={(1,a),(2,c),(3,b)}   S={(a,B),(b,A),(c,C)}

If the second element of the pair of each member of R maches the first element of a pair of a member of S, we can take the first element from the pair in R and the second element from the pair in S to create the pair in T.

T=S∘R={(1,B),(2,C),(3,A)}

So the element b is the link between the members of R and S.
