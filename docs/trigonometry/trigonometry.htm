The basic relations

Introduction

Let's consider an ordinary triangle:

The sum of angles in a planar triangle is 180 degree, which is π in radians:

α+β+γ = π

Rectangular Triangles

The point C lies on a circle with radius r around the mid-point of AB.

c = AB = 2r

The fundamental relationships in a rectangular triangle are

sinα = a c cosα = b c tanα = a b

Additional Relations

Some less frequently used relations are secans and the cosecans:

secα = cb = 1cosα cscα = ca = 1sinα
