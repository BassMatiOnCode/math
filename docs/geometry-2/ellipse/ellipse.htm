The Ellipse

Sections of a cone

Introduction

Originally, the ellipse was defined as an angled section of a cone, and appears as a circle, looked from above at an angle.

Axes

An ellipse has a center point C, and two orthogonal axes, the major axis AA with a length of a, and the minor axis mtext class="overline">BB with the length b.

Note that in an ellipse, the major axis is always longer than the minor axis (hence the name):

a>b (1)

If a=b, the ellipse is degraded to a circle.

The end points A1 and of the major axis are called vertices, the end points of the minor axis are called co-vertices.

The focal points lie always on the major axis, never on the minor axis.

Excentricity

The length c is called linear excentricity and is defined as:

c = a2 − b2 (2)

with

0≤c<a

The ratio

ε = ca = a2 − b2 a = 1 − b2 a2 (3)

is called numeric excentricity or simply excentricity, with

0≤ε<1

If b=c, we have a perfect ellipse:

a2 = b2 + c2 = b2 + b2 = 2b2 12 = b2 a2 1 2 = b a ε = ca = ba = 1 2

Gärtner's Rule

For each point P=(x, y) on the ellipse, the sum of distance to the focal points PF1+PF2 is equal to the length of the major axis A1A2. This is the first definition of an ellipse:

PF1 + PF2 = 2a (6)

This is shown in figure 5a:

For the following examinations, we assume that a>b to avoid collapsing to a line, and a>0 to prevent collapsing to a point.

The idea is to express 2a with a combination of a and c:

F1P + F2P = A2F1 + A2F2 = (a+c) + (a−c) = a+a

This might be a little bit confusing at first, but if you consider the lines as vectors in 2D space, it makes perfect sense. Walk from F1 over P, F2, C, A2, C back to the starting point F1, and add the length values.

Then we use the definition of the linear excentricity in equation (2), which exploits the theorem of Pythagoras:

c = a2 − b2

To simplify the math, the following text assumes that the ellipse is centered at the origin of a cartesian coordinate system. But the strategy works for every other location, too.

If we have the coordinates of the points, we can calculate distances:

F1P + F2P = (c+x) 2 + y2 + (c−x) 2 + y2 = 2a

The goal is to transform this equation into an expression with x, y, a and b.

The first step is to separate the radicals, then square both sides:

(c+x) 2 + y2 = 4a2 − 4a (c−x) 2 + y2 + (c−x) 2 + y2

Then solve the square of sums:

c2 + 2cx + x2 + y2 = 4a2 − 4a (c−x) 2 + y2 + c2 − 2cx + x2 + y2

Subtract equal terms from both sides, divide my minus 4:

4cx = 4a2 − 4a (c−x) 2 + y2 − 2cx

The next step isolates the radical:

a2 − cx = a (c−x) 2 + y2

Getting rid of the radical by squaring both sides:

a4 − 2a2cx + c2 x2 = a2 ( (c−x) 2 + y2 )

Solving the squared bracket:

a4 − 2a2cx + c2 x2 = a2 ( c2 − 2cx + x2 + y2 )

Distrubute a2:

a4 − 2a2cx + c2 x2 = a2 c2 − 2 a2 cx + a2 x2 + a2 y2

Subtract identical terms:

a4 + c2 x2 = a2 c2 + a2 x2 + a2 y2

Reorder the terms:

c2 x2 − a2 x2 − a2 y2 = a2 c2 − a4

Factoring out:

x2 ( c2 − a2 ) − a2 y2 = a2 ( c2 − a2 )

Divide by the entire right side:

x2 ( c2 − a2 ) a2 ( c2 − a2 ) − a2 y2 a2 ( c2 − a2 ) = 1

The previous step introduces two conditions:

  1. a2≠0
  2. c2 − a2 ≠0

Remove the canceling factors:

x2 a2 − y2 c2 − a2 = 1

This already looks pretty nice. We still need to get rid of the c. So we use the definition of the linear excentricity

c2 = a2 − b2 c2 − a2 = − b2

Replace that term, and we obtain the general equation for the horizontally oriented ellipse, centered at the origin:

x2 a2 + y2 b2 = 1 (7)

If the ellipse is centered at a different position (x0 , y0), the equation changes to

( x − x0 ) 2 a2 + ( y − y0 ) 2 b2 = 1 (8)

For the vertically oriented ellipse, a and b simply exchange places, because a is now a y quantity, and b is now an x quantity:

( x − x0 ) 2 b2 + ( y − y0 ) 2 a2 = 1 (9)

Tangent

The tangent equations for the vertices of an ellipse in horizontal orientation, with focus points on the x axis, are

x=±a y=±b

For any other point on the ellipse, we can differentiate the general equation of the ellipse. The method is implicit differentiation:, which gives us:

y = 2x a2 + 2yy′ = 0 → y′ = − x y b2 a2

Which gives us the point-slope form of a tangent line at P=(x0 , y0).

y = − x0 y0 b2 a2 ( x − x0 ) + y0

Solved for x:

x = − y0 x0 a2 b2 ( y − y0 ) + x0

With

x02 a2 + y02 b2 = 1

follows

x02 a2 x + y02 b2 y = 1 (10)
