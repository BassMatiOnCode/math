Parametric Representation

Introduction

The construction method of Phillipe de la Hire uses the two vertex circles of the ellipse, with r1=a=|CQ| and r2=b=|CR| to construct the point P on the ellipse:

axsecΘ − bycscΘ = a2 − b2

The point Q determines the x-coordinate of the point P and has the coordinates (acosτ,asinτ).

The point R determines the y-coordinate of the point P and has the coordinates (bcosτ,bsinτ).

So the point P has the coordinates (acosτ,bsinτ).

NOTE that τ and φ are NOT identical. The relation between τ and φ in a not rotated ellipse with a > b is

tanφ = ba tanτ = 1 − ε2 tanτ

Interpretation: If we multiply the y-coordinate of P by a/b, we get to the point Q.

yq = ab y = ab xtanφ = xtanτ = asinτ

We can (???) inject these expressions into the ellipse equation:

x2 a2 + y2 b2 = 1 = a2 cos2τ a2 + b2 sin2τ b2

to obtain the parameterized ellipse equation:

cos2τ + sin2τ = 1
