Introduction

Postulates of Euclidian Geometry

  1. Straight lines from any point to any other point
  2. Extend a finite straight line continuously in a straight line
  3. Describe a circle with any center and distance (radius)
  4. All right angles are equal to one another
  5. The parallel postulate: If a line crosses two straight lines, these meet at the side where the angles between the crossing line and the straight lines are less than two right angles.

Line Equations

General Equation of a Line

ax+by+c=0 (1)

Slope-Intercept Form

With x0 and y0 being the intersection points of the line with the x and y axes, we have for a non-vertical line:

y=mx+y0 (2)

For a non-horizontal line, it is

y=m(x−x0) (3)

These forms can be derived from the general line equation by solving for y:

ax+by+c=0 by=−ax−c
y=−abx−bc (4)

Comparing the factor in front of x reveals

m=−ab (5)

Setting x=0 in equation (4) gives the value for y0:

y0=−bc (6)

Solving (1) for x:

ax=−by−c x=−bay−ca

Then setting y=0 gives the value for x0:

x0=−ca (7)

Intercept Form

x x0 + y y0 = 1 (8)

If you set x=0 or y=0, you get the intercept values y0 and x0.

Point-Slope Form

A non-vertical line can be defined by its slope m and the coordinates x1 and y1:

y = y1 + m ( x − x1 ) (9)

Slightly rearranges, this form emphasizes the fact that the slope of the line can be computed from two points on the line:

y − y1 = m ( x − x1 )

By setting x=x2 and y=y2:

m = y2 − y1 x2 − x1 (10)

Two-Point Form

There is exactly one line that passes through two given distinct points P1 and P2. By combining equations (9) and (10) we get

y = y1 + y2 − y1 x2 − x1 ( x − x1 ) y − y1 = y2 − y1 x2 − x1 ( x − x1 )
( y − y1 )( x2 − x1 ) − ( y2 − y1 )( x − x1 ) = 0 (11)

Isolating the constant terms:

y( x2 − x1 ) − y1 x2 + y1 x1 = ( y2 − y1 ) x − x1 y2 + x1 y1 y( x2 − x1 ) − ( y2 − y1 ) x = x2 y1 − x1 y1 − x1 y2 + x1 y1 ( y1 − y2 ) x + ( x2 − x1 ) y = x2 y1 − x1 y2

gets us to a symmetric form in two given points:

( y1 − y2 ) x + ( x2 − x1 ) y + x1 y2 − x2 y1 = 0 (12)

Determinant Form

The two-point form (11) suggests to write a determinant:

| x − x1 y − y1 x2 − x1 y2 − y1 | = 0

Equation (12) can be derived by developing the determinant of this matrix after the first row:

| x y 1 x1 y1 1 x2 y2 1 | = 0

Intersection of Lines

Point-Slope Form

Suppose we have two lines:

y = y1 + m1 ( x − x1 ) y = y2 + m2 ( x − x2 )

The intersection is the point where y is equal for a certain x:

y1 + m1 ( x − x1 ) = y2 + m2 ( x − x2 )
