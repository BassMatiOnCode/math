Propositional Logic

Propositional or sentential logic is a simple formal system of reasoning.

Propositon

Definition: A proposition is a statement which resolves to the value of either true, or false. It is the smallest possible entity in propositional logic. Examples:

A = "It is cold outside"
B = "The sun is shining"

A proposition can evaluate to either true, or false, but never both (law of non-contradiction).

A question is not a proposition, a question cannot be evaluated to true or false.

Propositions are represented by captial letters. Lowercase letters are used for general propositions that have no meaning. They are used for general proofs.

Examples:

20x + 10y is not as statement because it contains variables. It has the form P(x,y)

The sun shines is a statement because it contains no variables. It has the form P.

Connectives

Connectives "connect" or "are connected" to propositions. Examples for connectives are

  • not
  • and
  • or
  • if-then
  • if-and-only-if (iff)

Examples:

A = The sun shines, B = It is cloudy.

not A means "the sun does not shine".

A and not B means "the sun shines and it is not cloudy".

If not A then B means "if it is not cloudy, then the sun shines".

Operators

Negation

b = ¬ a

Conjugation

Conjugation is the logical "and":

x = a ∧ b
Commonly used symbols: ∨ & * ⋅ ×

Disjunction

Disjunction is the logical inclusive "or":

x = a ∨ b
Commonly used ymbols: ∨ +

Implication

Implication is the logical "if-then":

x = a ⇒ b
Commonly used symbols: ⇒ →

Equivalence

Equivalence is similar to "equal", but not identical.

x = a ⇔ b
Commonly used symbols: ⇔ ↔

Predicate Logic

Predicates

A predicate is a symbol that represents a property or a relation. For instance, in the first-order formula P(a), the symbol P is a predicate that applies to the individual constant a. Similarly, in the formula R(a,b), the symbol R is a predicate that applies to the individual constants a and b.

A predicate is a function from the domain of objects to the truth values "true" and "false".

Predicates are interpreted as relations between variables. They are non-logical symbols.

Values

There are only two logic or boolean values:

true 1 T Yes
false 0 F No
