Sets can also be defined by one or more conditions. For example, the set of all even numbers can be defined as

A={x|x∈ℕ∧xis divisible by2}

This is read as "A is the set of all x (such that | with) x is an element of the natural numbers and x is divisible by 2". Sometimes a colon is used for the vertical bar, and the initial superset membership is on the left side of the bar:

A={x∈ℕ:xis divisible by2}
