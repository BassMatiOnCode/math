The Binomial Theorem

The binomial coefficients give us the number of different ways to choose n elements from a set of k elements. They also appear as coefficients in the general binomial formula.

Definition of the Binomial Coefficient

The binomial coefficient, read as "n over k" or "n choose k" is defined as follows:

NOTE that 0!=1.

These coefficients appear in Pascal's triangle:

n=0 1
n=1 1 1
n=2 1 2 1
n=3 1 3 3 1
n=4 1 4 6 4 1

Observe that each entry is the sum of the two entries above:

Proof: We plug the definition of the binomial coefficient into the hypothesis L1. The left-hand side then becomes

LHS= (n+1)! (n+1−k)!k!

The same with the right-hand side:

RHS= n!(n−k+1)!(k−1)! + n!(n−k)!k!

We will combine these terms over a common denominator. We need to multiply the first term by k, and the second term by n−k+1, and then the k terms cancel out:

= n!k + n!(n−k+1) (n−k+1)!k! = n!(n+1) (n−k+1)!k!

Then we rearrange the terms in the denominator, and thereby obtain the LHS of our hypothesis:

RHS= (n+1)! (n+1−k)!k! =LHS

q. e. d.

This lemma can now be used in the proof of the binomial theorem:

NOTE that we can use row n in Pascal's triangle to quickly find out the coefficients of the terms in the binomial formula.

PROOF: We will use inducion on n.

1. For n=0, the hypothesis' LHS=1, and the

RHS= (00)x0y0 =1 =LHS

So the hypothesis is true for n=0.

2. Inductive hypothesis: We suppose the hypothesis is true for some arbitrary n, then we suppose the hypothesis is true for n+1:

LHS= (x+y)n+1 =(x+y)(x+y)n =(x+y) ∑ k=0 n (nk) xk yn−k

Distribute the sum expression into the parentheses:

= x ∑ k=0 n (nk) xk yn−k + y ∑ k=0 n (nk) xk yn−k = ∑ k=0 n (nk) xk+1 yn−k + ∑ k=0 n (nk) xk yn+1−k

Then we will take out the final term from the first sum, and the first term of the second sum:

= xn+1 + ∑ k=0 n−1 (nk) xk+1 yn−k + yn+1 + ∑ k=1 n (nk) xk yn+1−k

Note that the index in the sum expressions has a scope that is limited to the sum expression, so we can rephrase the index in one sum without affecting the other. Of course, rephrasing must not change the value of the sum in any way. The goal is to have an idential index range in both sum expressions so that we can combine the sums:

= xn+1 + ∑ k=1 n ( n k−1 ) xk yn+1−k + yn+1 + ∑ k=1 n (nk) xk yn+1−k =xn+1 +yn+1 +∑k=1n [ (nk−1) xk yn+1−k +(nk) xk yn+1−k ]

Now we can use lemma L1 to simply the binomial factor expressions, which then paves the road for further simplifications. The two binomial coefficients in Pascal's triangle can be replaced by the one below them:

=xn+1 +yn+1 +∑k=1n (n+1k) xk yn+1−k

Now we can integrate the powers of x and y into the sum, they fit into the sequence if we adjust the range for the indexer. For k=0, the sum expression is (n+10)x0yn+1 = 1⋅1⋅yn+1, and for k=n+1, the sum expression is (n+1n+1)xn+1y(n+1)−(n+1) = 1⋅xn+1⋅1.

=∑k=0n+1 (n+1k) xk yn+1−k =RHS

Which is identical to the right-hand side, which we wanted to show.

3. By induction, the result holds for all n∈ℕ.

q. e. d.
