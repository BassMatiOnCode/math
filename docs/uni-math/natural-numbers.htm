The Theorem of Weak Induction

These induction theorems are a cornerstone of mathematical proofs. The key idea is to verify a hypothesis for the first (e. g. smallest) element in the set, usually zero or one. Then we suppose that the hypothesis is true for any arbitrary (unspecified) natural number n and show that the hypothesis holds for its immediate successor, n + 1. So if the hypothesis is true for n = 0, it must be true for n + 1 = 1. Then if the hypothesis is true for n = 1, it must be true for n + 1 = 2. And so on, infinitely. Thus we have proven that the hypothesis holds for all elements in the set.

We begin with the simpler weak induction theorem:

An example:

Proof:

1. Suppose P(0)istrue:

We set n=0 and calculate the left hand side of the hypothesis:

LHS=∑k=00k=0

Then the same with the right-hand side of the hypothesis:

RHS= 12 ⋅0⋅ (0+1) =0

LHS equals RHS, so the hypothesis holds for n=0.

2. Suppose that if the hypothesis is true for n, it is also true for n+1:

LHS=∑k=0n+1k = ( 0+1+2+…+n)+(n+1)

This is exactly the sum of the first n term plus an additional term n+1:

= ∑k=0nk +(n+1)

which is, according to our hypothesis, equivalent to

= 12 ⋅ n ⋅ (n+1) + (n+1)

Now we will check the right-hand side of the hypothesis by replacing n with n+1:

RHS= 12 ⋅(n+1) ⋅ ((n+1)+1) = 12 ⋅ (n+1) ⋅ (n+2)

Now we show that LHS = RHS:

12⋅n⋅ (n+1)+(n+1) = 12 ⋅ (n+1) ⋅ (n+2)

Dividing by (n+1):

12⋅n+1 = 12 ⋅ (n+2) = 12 ⋅ n+1

Both sides are identical, therefore the hypothesis is true for n+1.

3. Therefore the hypothesis is true for all n∈ℕ.

q. e. d.

The Theorem of Strong Induction

The weak induction theorem chooses an arbitrary element n and verifies the hypothesis for the immediate successor, n + 1. There are situation where this is not sufficient. Instead, there are situations where the hypothesis must be proven for n and all its predecessors up to the least element in the set. This type of proof is called the strong induction theorem:

Proof:

We will transform this theorem into the form of the weak induction theorem, which already has been proven. So define Q(n) to be the statement "P(k) is true for k=0,1,…,n". Then we

1. We know that Q(0) is true because P(0) is true.

2. For any n∈ℕ, if Q(n) is true, then Q(n+1) is true.

Now we have the exact same form as in theorem 1.

3. So, by induction, Q(n) is true for all n∈ℕ.

Hence, P (n) is true for all n∈ℕ.

q. e. d.

An application example:

Proof: Let P(n) be the statement that n can be expressed as a product of primes.

1. P(2) is true, because 2 is prime.

2. Let n>2, and suppose that P(m) holds for all m<n:

If n is prime, then P(n) is true. If n is not prime, then n=r⋅s for some r,s∈ℕ,1<r<n,1<s<n.

By the inductive hypothesis, r and s can be expressed as product of primes, and hence can n=r⋅s. So P(n) holds.

3. By strong induction, P(n) holds for all n∈ℕ.

q. e. d.

Another proposition:

We will combine induction with contradiction.

Proof: Assume, for a contradiction, that S is a non-empty subset of the natural numbers that does not have a least element. Consider a set R as a complement of S that contains all elements that are not in S: R={n∈ℕ:n∉S}.

Now we consider 0, which is the least element of the natural numbers, according to the definition of the natural numbers in D1. According to our assumption, 0 must be in R because S doesn't have a least number.

If 0,1,…,n∈R, then n+1∈R, because otherwise it would be the least element of S.

By strong induction, every natural number n∈R. Therefore S=∅, which contradicts our assumption. Thus the hypothesis is true.

q. e. d.

In this proof we haven't spelled out explicitly what the family of statements P(n) is, but that should be clear from the context (n∈R).

The well-ordering property of the natural numbers is quite important, in fact it almost defines the natural numbers.

Here we used induction to prove the property, but we could have gone the other way round and use the well-ordering property to prove the principle of induction.

The Binomial Theorem and an Introduction to Sets

The binomial coefficient gives us the number of choices to choose n element out of a set of k elements.

NOTE that 0!=1.

These coefficients appear in Pascal's triangle:

n=0 1
n=1 1 1
n=2 1 2 1
n=3 1 3 3 1
n=4 1 4 6 4 1

Observe that each entry is the sum of the two entries above. This is the content of the following lemma:

Proof: It is just a matter of plugging in the definition:

RHS= n!(n−k+1)!(k−1)! + n!(n−k)!k!

We will combine these terms over a common denominator. We need to multiply the first term by k, and the second term by n−k+1, and then the k terms cancel out:

= n!k + n!(n−k+1) (n−k+1)!k! = n!(n+1) (n−k+1)!k! = (n+1)! (n+1−k)!k!

Then we must plug in the definition into the LHS of the hypothesis:

LHS= (n+1)! (n+1−k)!k! =RHS

Which is obviously identical to the right-hand side.
