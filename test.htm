Vector Algebra

Space

Let't talk about space first. We all have a trivial notion of space, 1D, 2D and 3D space. And even 4D, if we add time. So a space has a number of dimensions. A dimension is represented by an axis of a coordinate system. We define a unit length on each axis so that we are able to identify the location of every point in space by a sequence of numbers, so-called coordinates.

Coordinate values are counted positive in axis direction, and negative in the opposite direction.

Note that the dimensions are independent of each other. We can in move in the direction of a coordinate system axis so that exactly one coordinate changes, while the others remain constant.

So a point in an n-dimensional space of real numbers is represented by an ordered set of n real numbers:

P∈ℝn , a∈ℝ , P= ( a1 a2 ⋮ an )

The coordinate system axes meet in the origin under defined angles:

Usually a cartesian coordinate system is choosen, where the axes are orthogonal to each other (at 90 degree angle) and constitue a right-hand system: The thumb, index and middle finger of the right hand correspond to the x, y and y axes. Right-turns lead from one axis to the next: Viewing in z direction, the x axis is turned clockwise around z to get to the y axis. Alternatively, consider turning an ordinary screw driven into the ceiling. Looking in the direction of x, rotating the y axis clockwise around x gets us to the z axis. And so on.

Note that if you look in the opposite directions (which seems to be a more "natural" point of view), you will have to counter-clockwise rotate instead.

The origin is considered "point zero" for the axes.

Each pair of axes spans a plane. In a 3D cartesian coordinate system, we have three planes, each one orthogonal to the others.

Points

A point has a location in space, but no direction:

With a given coordinate system, the location can be uniqely specified with an ordered set of coordinates:

P1= ( ax , ay ) P2= ( a1 , a2 , a3 )

Points are usually written in uppercase letters. For identification purposes, a number can be appended or subscripted. The set of coordinates is enclosed in rounded parentheses, and arranged horizontally (in contrast to vectors, which are usually arranged vertically).

If ai are real numbers, P2 is considered a point in the three-dimensional space of real numbers:

P2∈ℝ3

A coordinate can be considered the distance of the parallel projection of the point on the axis from the origin.

Remember to use a parallel projection to obtain the correct coordinate values. This is especially important if the axes are not orthogonal to each other. So don't blindly raise a perpendicular on the coordinate system axes. Instead, use parallel lines:

So, to obtain ax, draw a line parallel to the y axis through P1 down to the x axis. To obtain ay, draw a parallel to the x axis through P1 left onto the y axis. The concept also applies to higher dimensional spaces.

Scalars

A scalar is just another word for "number" or "value". In contrast, a vector is a combinations of a number (which represents the length of the vector), and a direction.

Vectors

In constrast to a point, a vector represents a distance or an offset. A vector has a direction, but no location.

From a geometrical point of view, a vector is an object which has a direction and a length or magnitude. In contrast to a point, a vector has no location and thus can be moved in space as long as its direction is maintained.

The parallel projection of the vector a⃗ = OP on the coordinate system axes produces the vector components ax, ay, and az. They can be considered the lenght of the vector in the direction of the coordinate system axes, and are real numbers or scalars. To distinguish a vector notation from a scalar, the vector is typed in a bold face or has an arrow above.

From an arightmetic point of view, a vector is defined by an ordered sequence of its components, also called coordinates:

a→=(ax, ay) b→=(bx ,by, bz) c→=(c1 ,c2, c3)

The amount of coordinates is equal to the number of dimensions of space. For example, in the two-dimensional space R2 we need two coodinates to specify the location of a point, in the 3D space R3 we need three coordinates.

The component subscript is sometimes a letter to indicate the relation to a named coordinate axis, sometimes just a number.

To discern a vector from a scalar, the vector has an arrow above or is written in bold face. A vector in component form can be written with parentheses, sometimes horizontally, but most of the time vertically:

a→ = a = ( a1 a2 a3 )

Identity

Two vectors are identical if they have the same direction and the same length.

Because a vector is defined by its scalar components, the definition above is equivalent to

Two vectors are identical if their scalar components are identical.

Parallel

Vectors are parallel if they have the same direction and orientation. They are anti-parallel if they have the same direction but opposite orientation. Multiplying a vector with the scalar -1 changes the orientation (inverts the direction).

Commutativity

The vector addition is commutative:

a→+b→ = b→+a→

Zero Vector

The neutral operand of the vector addition is called the zero vector, so that

a→+0→=a→

The zero vector has a zero length, and thus no orientation or direction.

Scalar Dot Product

The dot product or scalar product or Euclidian innner product operator is a multiplication operator that takes two vectors and returns a scalar.

Geometrically, the dot product of two vectors is a→·b→ is the orthogonal projection of a→ onto the vector b→, multiplied with the length of b→:

This can be expressed as

a→·b→ = |a→| · |b→| · cosφ = a·b·cosφ

Algebraically, the dot product is defined as sum of the component products:

a→ · b→ = ( a1 a2 a3 ) · ( b1 b2 b3 ) = a1 · b1 + a2 · b2 + a3 · b3

  • Check for parallelity (cos φ = 1)

The vector product × is an operator that takes two vectors and retuns a third vector. Geometrically, the result vector is orthogonal to the plane spanned by the operand vectors, and its length is calculated as
